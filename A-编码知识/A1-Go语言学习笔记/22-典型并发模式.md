# 22|典型并发模式

## 1.只运行一次

某些时候，我们需要某一段代码只需要运行一次，就可以使用这种模式

```go
package once_test

import (
	"fmt"
	"sync"
	"testing"
	"unsafe"
)

type Singleton struct {
	data string
}

var singleInstance *Singleton
var once sync.Once

func GetSingletonObj() *Singleton {
	once.Do(func() {
		fmt.Println("Create Obj")
		singleInstance = new(Singleton)
	})
	return singleInstance
}

func TestGetSingletonObj(t *testing.T) {
	var wg sync.WaitGroup
	for i := 0; i < 10; i++ {
		wg.Add(1)
		go func() {
			obj := GetSingletonObj()
			fmt.Printf("%X\n", unsafe.Pointer(obj))
			wg.Done()
		}()
	}
	wg.Wait()
}
```

运行结果如下

```

Create Obj

C000050180
C000050180
C000050180
C000050180
C000050180
C000050180
C000050180
C000050180
C000050180
--- PASS: TestGetSingletonObj (0.00s)
```

## 2.仅需任意一个完成

当某些时候有多个协程在处理任务，需要当其中一个任务完成后就返回结果的场景下，可以使用这种模式

```go
package concurrency

import (
	"fmt"
	"runtime"
	"testing"
	"time"
)

func runTask(id int) string {
	time.Sleep(10 * time.Millisecond)
	return fmt.Sprintf("The result is from %d", id)
}

func FirstResponse() string {
	numOfRunner := 10
	ch := make(chan string, numOfRunner)	//注意这里需要使用buffer模式，不然其余协程会进入阻塞
	for i := 0; i < numOfRunner; i++ {
		go func(i int) {
			ret := runTask(i)
			ch <- ret
		}(i)
	}
	return <-ch
}

func TestFirstResponse(t *testing.T) {
	t.Log("Before:", runtime.NumGoroutine())
	t.Log(FirstResponse())
	time.Sleep(time.Second * 1)
	t.Log("After:", runtime.NumGoroutine())

}
```

运行结果如下

```
=== RUN   TestFirstResponse
--- PASS: TestFirstResponse (1.01s)
    anyone_test.go:28: Before: 2
    anyone_test.go:29: The result is from 0
    anyone_test.go:31: After: 2
PASS
```

## 3.需要所有协程全部完成

方式一当然可以通过wait.group的方式等待所有协程全部完成，当然也可以通过CSP的方式解决这个问题

```go
package all

import (
	"fmt"
	"testing"
	"time"
)

func runTask(id int) string {
	time.Sleep(10 * time.Millisecond)
	return fmt.Sprintf("The result is from %d", id)
}

func AllResponse() string {
	numOfRunner := 10
	ch := make(chan string, numOfRunner)
	for i := 0; i < numOfRunner; i++ {
		go func(i int) {
			ret := runTask(i)
			ch <- ret
		}(i)
	}
	res := ""
	for j:=0;j<numOfRunner;j++{	//循环次数与协程数量一致
		res += <-ch + "\n"
	}
	return res
}

func TestAllResponse(t *testing.T) {
	t.Log(AllResponse())
	time.Sleep(time.Second * 1)
}
```
